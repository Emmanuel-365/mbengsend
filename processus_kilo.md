
Les Acteurs
Amine (Le Voyageur) : Voyage de Paris à Douala et veut rentabiliser ses kilos.
Béatrice (L'Expéditeur) : Veut envoyer un colis de 5kg d'Europe vers le Cameroun.
L'Équipe Mbengsend (Admin) : Coordonne, valide et sécurise l'opération.
1. La mise en ligne (Sur la Plateforme)
Amine se rend sur Mbengsend et remplit le formulaire "Vendre mes kilos". Il indique : Paris -> Douala, départ le 15 Avril, 20kg disponibles, il demande 8€ / kg.
Admin reçoit l'offre. Dans l'interface d'administration, il vérifie l'offre. Il décide de fixer le prix de vente à 12€ / kg (marge de 4€ pour Mbengsend). Il clique sur Approuver.
Résultat : L'offre apparaît instantanément sur le site pour tous les clients.
2. La Réservation (Sur la Plateforme)
Béatrice cherche à envoyer un colis. Elle voit l'offre d'Amine.
Elle clique sur Réserver des kilos, remplit ses informations (Béatrice, email, son numéro WhatsApp) et saisit 5kg.
Elle clique sur Confirmer. Elle voit le message : "Demande envoyée ! Mbengsend vous contactera."
Système : Une réservation est créée en base de données.
3. La Coordination (Hors Plateforme - WhatsApp)
Admin voit la nouvelle réservation dans son tableau de bord "Réservations GP". Il clique sur le numéro WhatsApp de Béatrice.
WhatsApp (Admin -> Béatrice) : "Bonjour Béatrice, nous avons reçu votre demande de 5kg pour Douala. Voici l'adresse de notre point de collecte à Paris. Pouvez-vous déposer le colis ce samedi ?"
Béatrice confirme le rendez-vous.
Admin contacte également Amine via WhatsApp pour l'informer : "Bonjour Amine, nous avons déjà 5kg réservés sur votre vol. Merci d'être prêt pour la collecte le 14 Avril."
4. Dépôt et Paiement (Physique / Offline)
Béatrice se rend au point de collecte Mbengsend, dépose ses 5kg.
Elle paie 60€ (5kg x 12€) via Cash ou Orange Money.
Admin ouvre son interface Medusa et met à jour la réservation :
Statut Paiement : Payé
Statut Colis : Colis reçu
5. Le Voyage et Remise (Physique)
Amine récupère les colis collectés par Mbengsend juste avant son vol.
Amine atterrit à Douala. Le destinataire (ou le point relais local) récupère le colis.
Admin marque la réservation comme Livrée dans le système.
Mbengsend verse à Amine sa part de 40€ (5kg x 8€).
